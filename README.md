## Clustering Algorithms Implementation and Comparison
### Overview

* This project implements and compares three popular clustering algorithms: 
  1. K-Means
  2. DBSCAN
  3. HDBSCAN

The goal is to understand how each algorithm behaves when applied to the same dataset and to visually compare their clustering performance, strengths, and limitations.

The dataset contains non-linear, irregularly shaped clusters with varying densities, making it suitable for evaluating both centroid-based and density-based clustering approaches.

### Installation
  1. Clone the repository:

    ```bash
    https://github.com/Leoul39/clustering-algorithms.git
    cd clustering-algorithms
    ```

  2. Create a virtual environment and install dependencies:

    ```bash
    python -m venv venv
    source venv/bin/activate   # On Windows use: venv\Scripts\activate
    pip install -r requirements.txt
    ```
  3. Run the jupyter notebook to see the plots
### Technologies Used
